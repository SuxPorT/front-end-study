Conhecimentos

HTML5

HTML5 é uma linguagem de marcação usada para estruturar e apresentar conteúdo na World Wide Web. É a quinta e mais recente versão principal do HTML que é uma recomendação do World Wide Web Consortium.

CSS3

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets (ou simplesmente CSS), pela qual se define estilos para um projeto web (página de internet). Com efeitos de transição, imagem, imagem de fundo/background e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página.

JavaScript

JavaScript, frequentemente abreviado como JS, é uma linguagem de programação que está em conformidade com a especificação ECMAScript. O JavaScript é de alto nível, geralmente compilado na hora certa e com vários paradigmas. Possui sintaxe entre colchetes, digitação dinâmica, orientação a objetos com base em protótipo e funções de primeira classe

Bootstrap

Bootstrap é uma estrutura CSS gratuita e de código aberto direcionada para o desenvolvimento web front-end responsivo e móvel. Ele contém modelos de design baseados em CSS e JavaScript para tipografia, formulários, botões, navegação e outros componentes da interface.

Linux

Linux é uma família de sistemas operacionais Unix, de código aberto, baseados no kernel do Linux, um kernel do sistema operacional lançado pela primeira vez em 17 de setembro de 1991 por Linus Torvalds. O Linux normalmente é empacotado em uma distribuição Linux.

Git e GitHub

Git é um sistema de controle de versão distribuído para rastrear alterações no código-fonte durante o desenvolvimento do software. Ele foi desenvolvido para coordenar o trabalho entre programadores, mas pode ser usado para rastrear alterações em qualquer conjunto de arquivos. Seus objetivos incluem velocidade, integridade dos dados e suporte a fluxos de trabalho não lineares distribuídos.
